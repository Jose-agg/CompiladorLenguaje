' 1) Metadatos para Funciones.
' El depurador incluye directivas para
' indicar el nombre, parámetros, 
' variables locales y tipo de retorno
' de las funciones.

' Directivas:
' #func <nombre>
' #param <nombre>:<tipo>
' #ret <tipo> / #ret void
' #local <nombre>:<tipo>
'
' Pueden colocarse en cualquier lugar
' del fichero (la información de cada
' función puede ir junto con su 
' código - antes o después - o bien
' se puede juntar la información de
' todas las funciones al inicio o final).
'
#func Calcula
#param numero:int
#param radio:real
#ret int
#local persona: {
 edad:int
 altura:int
}
